## Topic

**Heston Model & Stochastic Volatility** — Covers the Heston stochastic volatility model for option pricing, Monte Carlo simulation with the Heston process, and computation of option Greeks under stochastic volatility.

## Key Topics

- Heston stochastic volatility model (CIR process for variance)
- Monte Carlo simulation of the Heston process
- Cholesky decomposition for correlated Brownian motions
- Heston Call and Put option pricing
- Delta and Gamma under stochastic volatility
- Variance mean-reversion parameters (kappa, theta, sigma_v)
